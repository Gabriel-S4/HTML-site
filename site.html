Tipos de textos:

<h1> </h1> até <h6> </h6> → Tamanhos de cabeçalho / título e subtítulo

<p> </p> → Parágrafo

<blockquote> </blockquote> → Citação

<html> </html> → Tag de identificação HTML

<head> </head> → Tag de Cabeçalho (pode conter o título da aba)

<title> </title> → Título da página (aba)

<body> </body> → Tag de Corpo da página (contém o conteúdo da página)

<strong> </strong>→ Tag que deixa em Negrito (pode ser unida à tag que deixa em itálico)

<i> </i> → Tag que deixa texto em itálico

<u> </u> → Tag que deixa texto sublinhado (u de underline → _ )

<mark> </mark>→ Tag que deixa texto marcado (igual marca texto)

<sup> </sup> ou <sub> </sub> → Tag que deixa número/texto pequeno (parecido com um expoente → 123 ou a base de um logaritmo123)

<hr> </hr> → Tag que faz uma linha no meio da página (→ ——————— )

<small> </small> → Tag que faz um mini texto. (ex: texto)

<del> </del> → Tag que faz um risco no texto (texto)

<abbr> </abbr> → Mostra uma palavra por cima de outra, muitas vezes usado para explicar abreviações (Oms)

Atributos:

voltar

style=”alteração” → Formatar (trocar cor, fonte, direto no elemento) (CSS)

class=”titulo-principal” → Padronização de cor com (tudo que tiver relação com o título principal fica com uma cor parecida, é bom na organização) (CSS)

id=”identificação”

    input= "Entrada" → no input, se coloca o tipo de informação que o usuário pode colocar na página, sendo elas texto, número (type é específico do input)

  1. <input type=”text” /> → Texto (Geralmente usado em formulários)
  2. <input type=”number” /> → Só vai aceitar números
  3. <input type=”color” /> → Campo do tipo color (Seletor de cor)

    img= "Imagem"→ <img width="150" src=”endereço de uma imagem (arquivo, link de outra página)” /> → Adiciona uma imagem à página.

  • <img src="texto” → src (Source da imagem, link, arquivo, e outros)
  • <img width="150" src="texto” → width (Largura da imagem)

Links:

voltar

<a href=”link”> → Implementar link à um texto:

<a href=”Link.com”> Nome do texto que vai te endereçar ao Link

Target=”_self” → Escolha da maneira que a página do link vai se apresentar após ser clicada.

<a href=”www.algumsite.com” target=”_self”> Nome do texto que vai te endereçar ao Link

Tipos de target →

  • _blank → Abre o link em outra aba do navegador
  • _self → Substitui a página atual pela do link
  • _parent → Abre a página na mesma janela do link
  • _top → Cancela todos os demais frames e agre uma nova página no mesmo navegador
